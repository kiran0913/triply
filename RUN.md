# Travel Buddy Matching App

Modern, mobile-first web app for finding compatible travel companions. Built with React, Tailwind CSS, Lucide icons, and Framer Motion. Uses mock data for demo.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Pages

- **/** — Landing (hero, how it works, features, testimonials)
- **/login** — Login (email, Google)
- **/signup** — Sign up (email, password, Google)
- **/onboarding** — Multi-step profile setup
- **/dashboard** — Welcome, matches, trips, messages
- **/matches** — Swipe-style match discovery + filters
- **/profile** — Traveler profile
- **/profile/:id** — Other user profile
- **/chat** — Chat + trip planning panel
- **/create** — Create trip post
- **/explore** — Browse trips

## Build

```bash
npm run build
npm run preview
```
